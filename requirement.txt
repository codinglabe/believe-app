# Supporter Activity Tracking – Developer Specification

## Purpose

Create a simple analytics system to track *supporter engagement* across the platform.
The goal is to understand *which supporters are active and how they interact with organizations*.

Only *five events* will be tracked across the entire system. These events will power the *Supporter Activity Dashboard*.

---

# 1. Events to Track

The platform should only record the following events:


donation.completed
purchase.completed
courses.completed
events.completed
volunteer.signup


### Event Definitions

*donation.completed*
Triggered when a supporter successfully completes a donation.

*purchase.completed*
Triggered when a supporter purchases an item in the Marketplace.

*courses.completed*
Triggered when a supporter completes a course.

*events.completed*
Triggered when a supporter attends or completes an event.

*volunteer.signup*
Triggered when a supporter signs up for a volunteer opportunity.

---

# 2. Activity Tracking Table

Create a single table that stores all supporter actions.

### Table Name

supporter_activity

### Columns

| Column          | Type      | Description                                          |
| --------------- | --------- | ---------------------------------------------------- |
| id              | bigint    | Primary key                                          |
| supporter_id    | bigint    | ID of the supporter performing the action            |
| organization_id | bigint    | Organization involved                                |
| event_type      | string    | One of the five events listed above                  |
| reference_id    | bigint    | ID of related record (donation, event, course, etc.) |
| created_at      | timestamp | Time the action occurred                             |

# 6. Key Objective

This system allows the platform to measure:

* Which supporters are most engaged
* How supporters contribute (money, purchases, learning, volunteering)
* Recent supporter activity
* Overall supporter participation across organizations

This activity data will power the *Supporter Engagement Dashboard* used by organizations.
